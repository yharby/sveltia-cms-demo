# About This Site

This is a demonstration of **Sveltia CMS** - a modern, Git-based headless content management system.

## What is Sveltia CMS?

Sveltia CMS is the spiritual successor to Netlify/Decap CMS, built from the ground up with modern technologies:

- Built with **Svelte 5** for excellent performance
- **GraphQL API** for faster content loading
- **First-class i18n** support from day one
- **Mobile-optimized** interface with touch support
- **100% free and open source**

## Features Demonstrated

This demo site showcases:

- ✅ Blog posts with tags and images
- ✅ Static pages like this one
- ✅ Site settings management
- ✅ Markdown content editing
- ✅ Media library for images
- ✅ GitHub backend integration

## How to Use

### For Developers

1. Clone this repository
2. Customize `config.yml` for your needs
3. Add your OAuth credentials
4. Deploy to your favorite platform

### For Content Editors

1. Navigate to `/admin/`
2. Sign in with GitHub
3. Create and edit content
4. Save and publish
